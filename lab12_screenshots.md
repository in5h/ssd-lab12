# Lab 12 — Screenshot Checklist

Take the screenshots in order and include captions in your lab PDF. Filenames suggested below.

- `01_jenkins_dashboard.png` — Jenkins Dashboard (`http://localhost:8080/`).
- `02_new_item_multibranch.png` — New Item: create `myfirstpipeline` as Multibranch Pipeline.
- `03_branch_sources_config.png` — Branch Sources configured with GitHub repo `in5h/ssd-lab12`.
- `04_scan_repository_log.png` — Scan Repository Log showing Jenkinsfile detection.
- `05_branch_job_list.png` — Branch job list for `main` (or `master`).
- `06_pipeline_stages_view.png` — Pipeline stages view showing Build/Test/Deploy.
- `07_console_output.png` — Console Output showing `Building...`, `Testing...`, `Deploying...`, `Pipeline complete`.
- `08_build_with_parameters.png` — Build with Parameters screen showing `executeTests` boolean.
- `09_skipped_test_output.png` — (Optional) Console output showing Test stage skipped when parameter false.
- `10_global_tool_config.png` — Global Tool Configuration showing `Maven` and `JDK11` tool names.
- `11_env_vars_page.png` — Jenkins env-vars page (`/env-vars.html`) showing environment variables.
- `12_github_repo_jenkinsfile.png` — GitHub repo root showing the `Jenkinsfile` contents.
- `13_github_commit_history.png` — Commit history showing the Jenkinsfile commit.
- `14_github_webhook.png` — (Optional) GitHub webhook configured to `https://<host>/github-webhook/`.
- `15_ngrok_forwarding.png` — (Optional) ngrok forwarding URL if you exposed Jenkins.
- `16_pipeline_scan_errors.png` — (Optional) Any scan/build errors for troubleshooting.

Short caption for each: one line under the image describing what the screenshot shows.
